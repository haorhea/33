                        <div class="txt js-content ptag"><p>五月的第二个周日，花京院来到家中一百余天，这天是承太郎第一次在清晨睁开眼睛之后，发现身边的孩子不见了。</p> 
<p>他坐起身，揉眼，四处查看，掀起了被子寻找那个小小的身影，但是床上确实没有人，甚至由于他一开始的动静太大，整个床上都找不到花京院曾经睡在这里过的痕迹。</p> 
<p>五岁的花京院和十七岁的时候一样，平时虽然从不迟到，但实际上喜欢赖床，躺在他身边的时候就更容易睡不醒。只是现在这个孩子还没有过去花京院那种人前人后两种面貌的习惯，和承太郎独处的时候，也没有过分表现出恃宠而骄的情绪来，因此叫他起床稍微更容易些。然而无论如何，五岁的花京院典明也不像是会赶在承太郎之前早早起床的人，何况今天是周末，而时间还不到七点。</p> 
<p>监护人坐在床上叫他的名字，声音里带了点惶急，一声叫过之后就听到房间门口传来急急匆匆的应答声，花京院开门进来，手里拿着一张五颜六色的纸说，我在这里。</p> 
<p>承太郎松了口气，下床连拖鞋也没来得及穿，赤着脚走过去问道你大清早跑到哪里去了，花京院被他握住了肩膀，把那张花花绿绿的纸一个劲地往身后藏，表现得有点扭扭捏捏，承太郎的注意力自然被那东西吸引过去，然而花京院既然想藏起来，那他也能忍住了暂时不看。</p> 
<p>“我有东西要给你，承太郎，坐到床上去。”</p> 
<p>花京院扬扬下巴，语气里藏不住的兴奋与期待，显而易见，接下来显然有什么重要的事要发生了，而且同那张花纸有关。承太郎于是坐回床沿，等着孩子给他一个惊喜。花京院凑近过来又要他闭上眼睛，他乖乖闭上，听着面前半米的距离里纸张晃动铺展的响声，那张纸似乎很大。</p> 
<p>“好了，现在睁开眼睛。”</p> 
<p>他睁开眼睛的时候，第一眼看到的是图画，花京院给他画了一幅像，说实话，有些抽象。从成人的眼光看来，只能看出画面中的自己戴着帽子，穿着白衣，至于帽子上的徽章和五官长相之类的细节，看过就好，不能强求，花京院显然对他的眼睛颜色颇具执念，两个绿色的，大大的圈横亘鱼脸盘的中央位置，红色的线条则在绿头苍蝇一般的眼睛下面勾勒出三角形的鼻子和一张血盆大口。</p> 
<p>而视线向下扫去，接着映入眼帘的是两行写得歪歪扭扭的字：</p> 
<p>承太郎：我爱你！</p> 
<p>永永远远。花京院典明</p> 
<p>他愣了一下，把注意力从画转到人的身上，花京院捏着这张能把他整张脸都遮住的巨作，从纸张的上缘露出两只眼睛，颇为紧张地看着承太郎，显然在等他的评判。</p> 
<p>“谢谢你，我很喜欢。”他心想姑且先表示感激和喜爱吧，伸手拿过这张布满印痕的作品，花京院立刻明显地松下一口气来，他把画作和那句过于直接的表白交到承太郎手里，接着慢慢地打着哈欠挪上了床，一大早赶在承太郎之前醒来，他的确还泛着困。</p> 
<p>承太郎审视着手里那张薄薄的纸，眼睛在用黑色水彩笔写下的那行字上扫过一圈，又一圈。</p> 
<p>这有些突然。</p> 
<p>“为什么今天要给我这个？”</p> 
<p>“唔，”花京院的半个哈欠被扼杀在喉咙口，他停了一会儿，有些窘迫地说，“今天是母亲节。”</p> 
<p>这句话足以穿起所有的信息，让承太郎明白过来是怎么回事，他揉揉花京院的脑袋说是星期五在幼儿园里画的吗？花京院点点头，又跟着说，因为，我又没有妈妈。</p> 
<p>承太郎对自己的幼儿园时代多少还留存着一点印象，何况不止是幼儿园，就算是小学，老师也经常会在母亲节之类的时机让他们画贺卡，多半是一张拙劣的肖像，再加一句直白到令人脸红的“妈妈，我爱你！”他可以想象花京院在幼儿园也遇到了同样的事，老师虽然知道他情况特殊，却也不能为了他一人改变课程安排，更不能单独把这个孩子在这堂课请出去，没有妈妈的话，那么画父亲也可以的，毕竟向父母表达爱意，任何时机都不会错。怎么？也没有父亲吗？那么爷爷奶奶？另外其他的家人长辈？都没有吗？怎么会？那你家里总该有个人吧？</p> 
<p>我什么都没有，只有承太郎。</p> 
<p>这就是这张纸的来历了，拙劣但认真的画像，直接但并不是那个意思的表白，承太郎像当初收养花京院第一天的时候一样，把那张有些破烂的纸收进了文件夹，那文件夹如今已经很厚，搁在书房里，单独占据了目前还空空的一大格。他收好之后又顺手翻了翻花京院开始上幼儿园以后的作品，光是承太郎的画像就有好几张。其实幼儿园老师多半是布置了不同题目的，画一画妈妈，画一画爸爸，画一画你亲近的人，画一画你最喜欢的人，随便画一个人，诸如此类，花京院全都选择了承太郎。刚开始几幅整个画面都铺满了黑色，明明是温情的题目，画作却仿佛在描绘童话故事里的恐怖魔王，只差背后再加几笔张牙舞爪的火焰。而最近几幅开始，画中的人变成了白色，表情也总是灿烂到有点不符合现实的笑容，承太郎把最新一张放到末尾的夹层中，心想虽然都画得绿头苍蝇一般，但是竟然真的有明显进步。</p> 
<p>回到床边的时候，花京院已经重新睡了过去，斜倒在床上，脸埋进被窝，发出了细小的呼声。他给他重新盖上被子，心想，今天或许该给老妈打个电话。</p> 
<p>.</p> 
<p>七月，花京院和他都放了暑假，承太郎其实算是半个社会人，原本没有暑假这种说法，可是研究室的负责人不知怎么知道了花京院的事，信息出了点偏差，以为承太郎最近半年里在帮忙照顾一个亲戚的孩子，于是一直催他，你带孩子出去玩啊，你多陪陪她，你别担心研究室里的事，实验不会出差错。</p> 
<p>花京院住在他这里的事，原本承太郎也不觉得能藏住，毕竟作息时间猛然变化，又十分突兀地在某天改变了穿衣风格。换了白衣的第二天，整个办公室像是炸开了锅，好事之徒甚至都变得胆大包天，敢于凑上前来问他：空条，怎么回事？今天有约会？</p> 
<p>他皱眉，匆匆走过去，没有回答，直到吃午饭的时候才想起来今天原来是二月十四日，情人节。过往他是最烦这个日子的，无处不在的女人和巧克力，将他从教室围追堵截到校门口，逃都逃不过，高中时代火气最旺，收到了之后就当着她们的面，哐哐哐哐地将那些凝聚着心意的甜食一鼓作气扔进焚化炉，烧完了再气势汹汹地走掉，自以为这样做很帅气。上了大学之后，人变得沉稳了，或者说心如死灰，不再像高中的时候那样一肚子火气，巧克力和情书递过来，他多半把手插在口袋里说一句：对不起，我不能收。</p> 
<p>然而大概今天大约是改变太大，所有人被惊得心里一震，巧克力和告白，竟然一个都没有来，办公室里时常走动的几位女性，全都缩在原处探头探脑地揣测着动静。相对熟络又胆大的研究生在下午的实验间隙说，大家都以为空条是有了意中人又要在今天搞点什么大动静，这才换了这身虽然依旧帅气却非常诡异的白衣，所以目前女士们都按兵不动——倒不是因为他有了对象才按兵不动，而是他竟然会有对象，这事情冲击实在太大，这才吓得不敢随意出手。</p> 
<p>承太郎想到这里的时候忍不住笑了笑，心想说是为了意中人，倒也不算是说错。</p> 
<p>七月里顶着烈日到了金阁寺般大小的空条宅，赫莉先是无比热情地接待了花京院，等那热情稍微退散一些，于是便转而对承太郎的新造型大惊小怪。白色的帽子！她依旧掩着嘴，笑得像个十多岁的少女，上面还有海豚！</p> 
<p>花京院靠在这位伯母的怀里，在她善意地取笑她儿子的时候，勇敢而固执地一遍遍重复：我觉得很好看！</p> 
<p>正如承太郎的想象，花京院和赫莉处得很好，比起他自己，其实赫莉甚至乔瑟夫乔斯达都更像一个合适的监护人，他们总是对孩子有数不尽的温柔耐心，也有数不尽的奇怪点子来逗孩子们笑。花京院刚开始和他在一起的时候，虽然并不抵触，但总是端着一层淡淡的，成人式的拘谨，半年的朝夕相处才把这种孤儿院里带出来的拘谨从他身上剥离。然而和赫莉在一起的时候，他很快就露出无防备的样子，又笑又闹，两个年龄差了近半个世纪的人，几乎毫无隔阂，正很亲密地凑在一起说着悄悄话。</p> 
<p>他预计在这里住上四天，第二天是花京院的生日，第五天是研究室他主要负责的实验的阶段报告日，不回去实在不行。吃过晚饭之后他洗了碗，擦干手从厨房出来的时候，母亲正一手挽着花京院，一手给他看什么东西。</p> 
<p>“在看什么呢？”</p> 
<p>“你小时候的相册。”</p> 
<p>“你——唉，你真是——”</p> 
<p>“有什么关系，承太郎小时候也很可爱呀。”</p> 
<p>这大约是赫莉乔斯达表达亲热的方式：和人分享承太郎小时候的相册。外祖父的替身既然每次发动都要摧毁一台相机，那么乔斯达家显然是不缺照相录像设备的，承太郎的成长轨迹中落下了不少他不想让人见到的影像，幼儿园第一天回家之后抱着父亲的大腿大哭（空条贞夫在这张照片中只有大腿出境），小学入学仪式在校门口的留影，第一次参加远足，吃着母亲做的便当，对着镜头傻兮兮地笑……上了中学之后的稍微少些，但较之同龄人的留影数量，承太郎的照片之多仍旧算得上遥遥领先，赫莉当年若是有那个心思，完全能靠儿子的照片在附近几个区的女生之间大赚一笔。</p> 
<p>承太郎上了中学之后便开始抵触拍照了，走到哪里都被陌生人的视线捕捉就足够他烦躁，换了一个明目张胆的镜头，那感觉就更加讨厌。这习惯到他上了高中，去了埃及，也依旧是这样，某次见到外祖父架起相机却不似要进行念写，他下意识地压了压帽檐准备往吉普车后面躲，直到十七岁的恋人拉着他的胳膊说，好啦，到这边来，我们总该弄两张合照吧，他才勉勉强强地站到了镜头前，嘴里不情不愿地说，合照，回去再拍不也可以？</p> 
<p>二十三岁的空条承太郎解下了围裙放好，坐在桌子对面，母亲和花京院在看他的照片，而他看着他们，花京院不时从照片上抬起头来看看他，咬着下唇，憋着笑，承太郎也笑起来，问他怎么了？花京院哈哈哈地笑出声来说，照片上的承太郎和承太郎一点都不像。</p> 
<p>海洋博士生在那里坐了不足五分钟之后起了身，边走边说，你们慢慢看吧，我去洗澡。</p> 
<p>母亲和养子同时嗯了一声，注意力依旧钻在酒相簿里，没有抬头。</p> 
<p>他走去浴室，脱衣，擦洗，跨进浴池，最后泡在热水里闭上了眼睛，皮肤上一阵阵发烫的温度，恍惚间让他好像回到了六年前那个沙漠。那时候虽然是冬季，但白天的沙漠也是这样的热，花京院和他缩在越野车的后座，后座另一边还塞了个波鲁那雷夫，但他们两人挤在一起，法国人便不在考虑范围内了。花京院有些犯困，低着头，太阳穴一下一下地磕在他的肩膀上，承太郎这样看着他一会儿，最后伸手把他扶起来一些，结结实实地按在自己身边，花京院从浅眠中惊醒片刻，看到扶着自己的人是承太郎，犹豫半秒，默许了他的行为，头一歪，窝在他身上重新睡过去。</p> 
<p>这种他靠在自己身上的感觉，很真实，很令人安心。承太郎从这个角度看过去，恋人的耳坠，刘海，睫毛都随着越野车的行进微微跳动，而他的心随着这跃动轻轻地颤抖起来。他纵容自己继续放肆地盯着靠他这么这么近的花京院，心里想着，我真的好喜欢他啊，我想要他，想要他在我身边，想要带他回家。</p> 
<p>长长的车程中无所事事，聊天也只会惹得口干舌燥，除了负责当司机的人，其他的多半便闭目养神，承太郎则像个普通的，并不需要去拯救世界和母亲的十七岁少年一样，在颠簸中边睡边做着和恋爱有关的白日梦。他幻想着等这些事结束，他把花京院带回家，母亲必然和他相处得很好，而她若是喜欢这位未来的儿婿，必然就会掏出那些珍藏的旧相簿与花京院分享。花京院是个嘴很毒的人，虽然最常发作的对象不是他，但承太郎儿时的蠢照片也不少，那些哭笑得形象全无的黑历史被翻出来，又仗着岳母的宠爱，他肯定会嘲笑他一番，承太郎简直能活灵活现地想象出他们两个坐在客厅里，捧着那本封面全是小星星的相册的样子，母亲一一指点这是承太郎多大的时候在哪里做了什么，而花京院则看看照片又看看那时候必然十分窘迫还带了些恼羞成怒的他，抹着笑出来的眼泪说道：承太郎你小时候跟现在一点都不像！</p> 
<p>“你笑什么呢，”一阵瞌睡已经过去，抹着嘴角的花京院抬头看着他问道，“想什么呢？笑这么恶心。”</p> 
<p>“没什么。”十七岁的少年依旧在笑，这笑容停不下来，他想做点边的什么转移恋人的注意力。越野车开着窗，风沙很大，他眯起了眼睛，从花京院的脖子旁边捞起亚麻制的头巾往他头上胡乱地围过去，说，“没什么，一点小事，跟你有关的。”</p> 
<p>二十三岁的空条承太郎睁开眼睛的时候，温度依旧，可是没有风沙，眼前只有一缸滚烫的热水，蒸汽氤氲，连眼睛也有些潮了。</p>
